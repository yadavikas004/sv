# Server Virtualization (VMware vSphere 6.7) - Theory & Study Materials

This repository contains study materials and comprehensive guides for the **Server Virtualization** course, specifically focused on VMware vSphere 6.7.

## Contents

1. **[Definitions_and_Diagrams_InDepth.md](./Definitions_and_Diagrams_InDepth.md)**
   * Provides deep-dive explanations of core architectural concepts.
   * Includes structured Mermaid diagrams for topics like ESXi Architecture, vMotion, Memory Reclamation, and vSphere HA.
   * Highlights the exam probability of each topic.

2. **[Brief_Answers_QnA.md](./Brief_Answers_QnA.md)**
   * A rapid-fire, 10-mark structured Q&A guide.
   * Breaks down complex topics into 5-7 punchy, easy-to-memorize bullet points.
   * Includes a generic "Fallback" paragraph to ensure partial points if you get stuck on an exam question.

## Key Topics Covered

* **Unit I:** VMware ESXi Architecture, vCenter Server, SSO, DCUI.
* **Unit II:** Virtual vs Physical Switches, VMkernel Ports, vSAN, HA, Updates (VUM).
* **Unit III:** Virtual Machine Clones, Templates, OVF, Memory Transparent Page Sharing (TPS).
* **Unit IV:** vMotion features, Cross-vCenter vMotion, PowerCLI.

## Usage

These materials are highly optimized for university exam preparation and rapid revision.
